# ELITEMOTD Updates

This repository tracks version updates for the ELITEMOTD Minecraft plugin.

## Current Version

**1.0.0**

## How It Works

The plugin checks this repository's `version.txt` file on startup to determine if an update is available.

## Updating

To release a new version:
1. Update `version.txt` with the new version number
2. Plugin will automatically detect and notify admins
